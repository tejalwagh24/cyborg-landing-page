# 🤖 CYBORG-X Landing Page

A futuristic cyborg-themed landing page built using React and Vite, featuring a cyberpunk-inspired interface, glowing visual effects, and responsive design.

## 🚀 Features

* Futuristic Cyborg UI Design
* Responsive Layout
* Animated Hero Section
* Cyber Grid Background
* Neon Glow Effects
* Interactive Feature Cards
* Future Metrics Dashboard
* Modern CTA Section

## 🛠️ Tech Stack

* React
* Vite
* CSS3
* Framer Motion
* React Icons

## 📸 Preview

A cyberpunk-inspired landing page showcasing the future of human and artificial intelligence integration.

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
