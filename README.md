# 🤖 CYBORG-X — 3D Cyberpunk Landing Page

A futuristic **cyborg-themed interactive landing page** built using **React + Vite + React Three Fiber**, designed for TechFest submission.  
This project showcases a modern UI combined with a real-time **3D interactive experience**.

---

## 🚀 Live Demo

🔗 https://cyborg-landing-page.vercel.app

---

## 📌 Project Overview

**CYBORG-X** is a futuristic landing page concept representing the fusion of:
- Human intelligence 🧠  
- Artificial Intelligence 🤖  
- Cybernetic systems ⚡  

It features a **3D interactive object** embedded inside a cyber-themed UI to simulate a next-gen digital experience.

---

## 🎯 Features

✔ Cyberpunk-themed UI design  
✔ Fully responsive landing page  
✔ Interactive 3D model using React Three Fiber  
✔ Floating + animated 3D object  
✔ Glow / neon futuristic styling  
✔ Glassmorphism cards & sections  
✔ Smooth layout with modern UI structure  
✔ Tech-themed hero section with strong visual hierarchy

## 🧠 Tech Stack

- React (Vite)
- JavaScript (ES6+)
- CSS3 (Cyber UI styling)
- React Three Fiber
- Drei (3D helpers)
- Three.js

---

## 🧩 3D Feature

The project includes an interactive **3D Torus model**:
- Floating animation
- Mouse interaction (OrbitControls)
- Real-time rendering using WebGL
- Integrated into landing page hero section
